# 산업 사례 공통 → 빅맨 안전구역 시스템 IoT 센서(온도·진동) → RealSense + YOLO 엣지 게이트웨이 → NUC + BoT-SORT (worker ID) 수집 파이프라인 → team_id별 이벤트 수집 OpenSearch → OpenSearch (로그·집계·k-NN) 대시보드 → 팀 선택 대시보드 + 경보

위 대시보드 시연이 빅맨님 안전구역 관제 화면과 정확히 같은 형태입니다. 사례 1(벡터 RAG)은 k-NN 작업자 ReID, 사례 3(진동 예측정비)은 침입 패턴 분석과 같은 방향입니다.

⚠️ 출처·한계: 사례는 AWS OpenSearch 고객 페이지, AWS Industries 블로그, DigitalOcean·Smarttek 등 글로벌 사이트에서 확인한 것입니다. 위 대시보드·아키텍처 이미지는 실제 화면이 아니라 교육용으로 직접 그린 재현이며, 수치는 인용 사례의 대표값입니다. GE·Siemens 등은 자체 플랫폼 사용으로 제외했습니다.
