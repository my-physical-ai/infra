# 방향A 패키지 (4팀 통합 관제)

## 들어있는 것
- 4070_공용/        : 4070에 그대로 올리는 공유 시스템 (team_id 필터 대시보드 포함)
- make_team_nuc.sh  : 각 NUC에서 team_id 주입해 생성
- _src_nuc/         : NUC 원본 (스크립트가 복사해 사용)
- 방향A_가이드.md   : 전체 설명

## 빠른 시작
1. 4070_공용/ → 4070에 올리고 실행 (가이드 참고)
2. 각 NUC에 이 패키지 복사 후: bash make_team_nuc.sh [팀번호]
3. 모두 http://192.168.0.36:8500 접속 → 우측 "팀 선택"으로 구분

자세한 건 방향A_가이드.md 참고.
